# TOEIC Analysis Service

FastAPI-based IRT (Item Response Theory) analysis service for TOEIC test performance.

## Features

- **IRT Theta Calculation**: Maximum Likelihood Estimation of learner ability
- **Behavioral Analysis**: Speed, decisiveness, stamina, answer-change patterns
- **Weakness Detection**: Grammar, vocabulary, error pattern identification
- **Personalized Recommendations**: Actionable study advice

## Requirements

- Python 3.12+
- FastAPI 0.109.0
- NumPy, SciPy, Scikit-learn
